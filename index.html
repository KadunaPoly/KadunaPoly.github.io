Firefighter Robot

On-board tech in C++/C

This team project was offered by my school, the goal was to build a robot that could folow a path in order to map the room it travels. We were 6 teammates working on Linux and using Trello and Git in order to have a fluid teamwork. It was a huge project that took a whole semester to complete. That is why we separated the project in different parts. I worked on the function that were used to control the two motors of the robot. I had to implement function in order to make it go forward and turn both ways. Then i worked on the laser project. Indeed we were using 2 infrared lasers on boths sides of the robot. The goal was to determine the distance between the robot and the wall on each sides. I used physics equations to solve this problem. We later used this feature to map the room in which the robot was driving.

  • Date: autumn 2019
  • Language: C++/C
